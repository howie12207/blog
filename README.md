# 個人網站

### 概述

** 使用 Vite 建立專案，搭配 Vue 框架並以 Tailwind 做切版。 **

### 使用工具

- Vite
- Vue3
- Tailwind
- Element Plus
- Axios
- js-cookie
- vue-router

### 描述

文章與作品展示與個人介紹，可搭配後台做文章、作品 CRUD。
